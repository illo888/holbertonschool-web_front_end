# CSS Advanced

This project focuses on advanced CSS concepts and best practices, including:
- CSS variables
- Responsive typography
- Layout techniques
- Smooth scrolling
- Universal box-sizing
- Structuring large CSS files
- Working with normalize.css

## Files

Each `style-X.css` file builds on the previous one, adding new features and improvements.


## How to use
Simply open the HTML files in a browser and link the corresponding CSS files.

Example:
```html
<link rel="stylesheet" href="styles/15-style.css">
